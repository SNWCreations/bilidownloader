# Bilibili Downloader

无聊写的一个程序，基于 PyQt5 。

**警告: 此程序不支持在 Windows 以外的平台运行。**

本人一般不会对其进行维护，除非出现重大错误。

您可以自行构建 Windows 可执行文件，只需要运行 build.cmd (在 Windows 下) 即可。

    pyinstaller main.py -w -D -i="./icon.ico" --version-file file_version_info.txt

也可以直接运行 main.py 。
